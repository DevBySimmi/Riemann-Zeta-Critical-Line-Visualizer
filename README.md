## Mathematical Background

The Riemann Zeta Function is defined by

ζ(s) = Σ (1 / n^s)

for complex values of s.

It also has a remarkable connection to prime numbers:

ζ(s) = ∏ 1/(1 - p^(-s))

where p runs over all prime numbers.

The Riemann Hypothesis states that every non-trivial zero of ζ(s) lies on the critical line:

Re(s) = 1/2
